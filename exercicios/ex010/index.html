Para adicionar link ao texto/página em HTML usamos a tag < a >< /a >, onde ela cria a ancora entra o link externo e a página que criamos.
Dentro da tag podemos colocar os parâmetros: "target" e "rel".
Em target usamos o "_black" para abrir o link em uma nova aba e o "rel" para indicar que é um link externo"

Receita para lasanha bolonhesa.

OBS: O link fica envelopado dentro do texto em que selecionamos, ao clicar nesse texto a nova aba é aberta.

Links Internos

Link internet serve para criar uma ligação entre links/páginas do mesmo repositório. Diferente do link internet onde colocamos a link de algum site no internet solocamos o caminho da página dentro do respositório. Usamos também a tag < a > < / a >, e os parametros "rel" e "target" a diferença é que em target usamos o parametro "_selfie" para identificar que o link contine na mesma página, em rel usamos o "next" para informar que estamos indo para a proxima página e o "prev" para informar que estamos voltando a página!
OBS. para selecionar uma página no mesmo repositório apertamos no teclado ctrl + espaço.
Já para adicionar uma subpasta desse mesmo repósitório apertamos ctrl+espaço e selecionamos a pasta em seguida a página.
obs2: Paara voltar da subpasta para a pagina inicial devemos colocar "../" antes do nome da primeira página!
