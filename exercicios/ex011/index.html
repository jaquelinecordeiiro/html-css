Adicionando imagens com ajuste de tamanho automatico

Para adicionar imagens que se ajustam ao tamanho da tela usamos a tag < picture >, dentro dessa tag colocamos a tag < img > com o s parâmetros: "type", "source" e "scrsel".
Onde source diz respeito ao tamanho que a imagem vai se ajustar (sempre colocar 50 a mais), type o tipo de imagem e scrsel o local em que a imagem esta armazenada.
