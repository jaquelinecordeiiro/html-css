Lista Ordenada

Para fazer uma lista ordenada usamos a tag << ol > < /ol >> , nela as listas ficam em ordem com caracteres na frente, pondenod ser numérico, ordem alfbetica ou números romanos, podemos alterar usando o parâmetro "type". Sempre do maior para o menor.
SEMPRE COLOCAR A TAG < < li > < /li >> na frente de todo paragrafo da lista!

Listas não Ordenadas

Para criar uma lista não ordenada usamos a tag < < ul > < / ul >. Ela vai criar a lista sem ordem crecente, somente com caracteres na frente. Podendo ser um circulo, circulo vazio e quadrado!

Exemplo lista não ordenata com parâmetro typ e tipo "circle"

  • massa de lasanha (pronta)
  • 500 g de presunto
  • 500 g de queijo mussarela
  • 500 g carne moída
  • 1 massa de tomate pronta
  • sal a gosto
  • pimenta-do-reino a gosto
