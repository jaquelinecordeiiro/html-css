FORMATAÇÕES ADICIONAIS

Destacar texto

Para criar marcador de texto usamos a tag << mark >> onde deixa o texto em destaque. Para alterar a cor da tag podemos adicionar um parâmetro nela < < mark style "background-color line;" >,
dessa forma altera apenas o texto selecionado.
Já para alterar todos os "marks" devemos colocar o parêmetro em "head" usando um seleto"

Letra grandes

Para deixar a letra grande usamos a tag <big> (não esta mais em uso)

Letra Pequena

Para deixar a letra pequena usamos a tag <smal> ainda esta em uso e remete a observações, informações adicionais, etc

Texto excluido

Usamos a tag <del> quando queremos colocar uma informação que possa ser lida mas que não deve ter relavância.

Texto sublinhado

Usamos a tag <ins> quando queremos sublinhar uma palavra ou alguma parte do texto.
Temos também a tag <u> que não é usada mais, porem ainda reconhecida pala IDE e HTML.

Texto sobrescrito

Para sobrescrever um texto, número ou letra, usamos a tag <sup>, como em 230.

Texto subescrito

Já para sobrescrever alguma palavra, número ou letra usamos a tag <sub> , como em k3
