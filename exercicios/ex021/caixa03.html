Grupos de tags

O Group Tags ou Grupos de Tags são junções de tags que ajudam a manter o site organizado.
Sendo elas:
HEADER, NAV, MAIN,SECTION, ARTICLE, ASIDE E FOOTER.

Com os grupos de tags podemos separar as aréas do site em:

  • HEADER: Aréa destinada ao cabeçalho.
  • NAV: Aréa destinada aos links de navegação que vão compor o site.
  • MAIN: Aréa de menu do site.
  • SECTION: Aréa determinada as seções de textos (Agrupamento temático de conteúdos).
  • ARTICLE: Aréa determina para artigos ou algum outro site relacionado ao mesmo assunto.
  • ASIDE:Aréa destina a conteudos como listas.
  • FOOTER: Aréa determinada para nota de roda pé.

    • OBS: é possivel colocar um grupo de tags dentro de outro!
