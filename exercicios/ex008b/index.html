Códigos

Para criar texto em formato, usando a fonte da letra, de código usamos a tag << code >>

Ex:
if (expressão booleana) { // código 1 } else { // código 2 } (sem a tag)

if (expressão booleana) { // código 1 } else { // código 2 } (com a tag)

Já para deixar com identação (espacendo e alinhamento estilo codigo) usamos a tag << pre >>

ex: 

            
                if (expressão booleana)
                    {
                        // código 1
                    }
                    else
                    {
                    // código 2
                    }

P.S PARA ALTERAR A IDENTAÇÃO (RECUO DO TEXTO) PODEMOS USAR A TECLA SHIFT (AUMENTA O RECUO) OU SHIFT + TAB (DIMINUI O RECUO!

Citações

Para fazer uma citação ("" - aspas duplas)no texto em HTML usamos a tag << q >>

ex: Deciframe mas não me concluas
